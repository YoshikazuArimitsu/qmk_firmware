# 有光式 Helix

* Eucalynベース
* 右Shift/右Ctrlは使わない派なのでよく使う記号に変更
* 開きカッコは使用頻度が高い(閉じカッコは補完任せであんま使わない)のでそれぞれ1キー割り当て
* 親指バックスペース便利
* ALBERTロゴ入り
* ちょっと気合入ったアニメーション入り(メモリが足りないのでデフォルトのキーマップを幾つか削除)
